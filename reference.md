## Ensō Features & Shortcuts

### Keyboard Shortcuts

| Shortcut        | Action                            |
| --------------- | --------------------------------- |
| `⌘ + S`         | Save note (export to `.md` file)  |
| `⌘ + C`         | Copy all text to clipboard        |
| `⌘ + K`         | Clear editor (with confirmation)  |
| `⌘ + Shift + C` | Toggle Coffee Shop Mode (privacy) |
| `⌘ + +`         | Increase text size                |
| `⌘ + -`         | Decrease text size                |
| `⌘ + 0`         | Reset text size                   |
| `⌘ + Shift + T` | Toggle Theme Picker               |
| `⌃ + ⌘ + F`     | Toggle Full Screen                |
| `⌘ + Q`         | Quit application                  |

### Writing Modes

- **Typewriter Mode** – Cursor forced to end, no text selection, auto-scroll
- **Coffee Shop Mode** – Privacy mode that replaces text with bullets (●)
- **Auto-hide UI** – Interface fades after 2 seconds of inactivity

### Text Options

- **Spellcheck** – Toggle on/off (default: OFF)
- **Autocorrect** – Toggle on/off (default: ON)
- **Autocapitalize** – Toggle on/off (default: OFF)
- **Text scaling** – Adjustable text size
- **Word count display** – Toggleable in View menu
- **Caret styles** – Bar, Underline, or Hidden

### Themes

**Light:** Black and White, Light, Sepia (default)  
**Dark:** Dark (default), OLED, Midnight

- Auto-switches based on system appearance
- Live preview in theme picker

### File Operations

- **Save** – Exports as `note-DD-MM-SS.md`
- **Copy** – Copies all text to clipboard
- **Clear** – Removes all content (with confirmation)

### Other Features

- RTL/BIDI text direction support
- Persistent settings via localStorage
- Settings reset option (Ensō > Clear All Settings)
- Help menu with features overview and contact links
